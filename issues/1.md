# Issue Title: Page shakes/moves on load making it unusable

## Problem Description
The entire page is shaking and moving immediately when the page loads in all browsers (Chrome, Firefox, Safari, Edge), making it unusable.

## Reproduction Steps
1. Open Index.html in any browser
2. Page loads
3. Entire page starts shaking/moving

## Expected Behavior
Page should load smoothly without any shaking or moving animations.

## Actual Behavior
The entire page is moving side to side, making it difficult to interact with or read the content.

## Environment
- Affected Browsers: All browsers (Chrome, Firefox, Safari, Edge)
- Issue Severity: High - Makes page unusable
- File: Index.html

## Possible Causes to Investigate
- CSS animations or transforms causing unwanted page movement
- JavaScript interval creating layout shifts
- Floating heart animation affecting page layout
- Overflow or positioning issues in CSS

## Additional Notes
The page contains floating heart animations and multiple JavaScript intervals that may be causing layout instability.
